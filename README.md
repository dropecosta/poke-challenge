# Poke Challenge
> Desafio para a vaga de Front-end Developer, que consiste no desenvolvimento de uma aplicação React.JS de acordo com os requisitos solicitados.

## Instalação

Instalando o Front-end:

```sh
npm install
```

Rodando o projeto:

```sh
npm start
```

Executando os testes:

```sh
npm test
```
